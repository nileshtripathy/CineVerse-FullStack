# 🎬 CineVerse

A full-stack movie ticket booking platform built using the **MERN Stack** (MongoDB, Express.js, React.js, Node.js). CineVerse allows users to explore movies, view details, select seats, and book tickets through a modern and responsive interface.

---

## 🚀 Live Demo

🔗 Frontend: https://cineverse-client-ashen.vercel.app/

🔗 Backend API: https://cineverse-server-inky.vercel.app/

---

## ✨ Features

- 🎥 Browse latest movies and show details

- 🔍 View movie information and schedules

- 🪑 Interactive seat selection system

- 🎟️ Ticket booking functionality

- 🔐 User Authentication & Authorization using JWT

- 📱 Fully responsive UI

- ⚡ RESTful API integration

- 🌐 Cloud database with MongoDB Atlas

- 🚀 Frontend & Backend deployed on Vercel

---
